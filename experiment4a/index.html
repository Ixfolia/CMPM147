Experiment 4a - Infinite Worlds




Translated from: infinite-worlds-forest


Description

An infinite world showing grassland, water and a forest slowly growing and taking over everything

Technical

When a tile is clicked, it places a patch of dirt on the ground. Trees gradually spawn across a grid using a using Perlin noise. Initially, a high treeSpawnThreshold limits tree presence, but this threshold decreases over time, allowing more trees to appear. It creates a natural and evolving landscape where tree growth is based on random noise.
