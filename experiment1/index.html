Experiment 1 - Getting Started

Translated from: dnd-quest-generator on glitch.com

Description

The main goal for this experiment is to create a quest generator for DnD campaigns. It Ggnerates quests for a dnd campaign. Can be used for if you're unsure of what to do when your players want to go on a quest.

Technical

  • A super simple quest generator that generates a quest for a dnd campaign.
  • Created a button that generates a new quest when clicked.
  • Created a button that generates a new quest for the next day when clicked [DOES NOTHING].
  • Gives the player a random reward when the quest is completed.
