Experiment 4c - Infinite Worlds


Description

What are you trying to acomplish with this experiment?
This simple generated world is used to simulate 3D terrain in a grassy environment.

Technical

I used Perlin noise to generate water tiles by using a noise threshold. Tiles above the threshold are water tiles and below are grass/land tiles. When generating water, it may have a chance to display a wave image momentarily. On land tiles, users can click on them to raise a block of dirt on it and changing the color to simulate a 3D elevation effect.

Reflection

Although this one is super simple, I really enjoyed trying to make the 3D blocks work. Though there are some flaws, I think it looks alright and actually looks like it's stacking blocks on each other.
